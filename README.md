# Inverted Search Engine (C)

## 📌 Overview
A file-based inverted index system implemented in C.

## ⚙️ Features
- Create inverted index
- Search words across multiple files
- Display word frequency
- Update database
- Save 

## 🛠 Technologies
- C Programming
- File Handling
- Data Structures (Linked List, Hashing)

## 🚀 How to Run
gcc *.c
./a.out file1.txt file2.txt
